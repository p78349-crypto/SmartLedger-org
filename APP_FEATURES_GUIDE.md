## 거래 및 수입

### ➕ 거래 추가 (Transaction Add Screen)
**경로**: `홈 → + 버튼` 또는 `계정 → 거래 추가`

**거래 입력 방식**:

**1️⃣ 간단 입력 (Quick Input)**
- 금액 입력
- 카테고리 선택
- 설명 추가
- 저장

**2️⃣ 상세 입력 (Detailed Input)**
- 거래 유형 선택 (입금/출금/반품)
- 금액 입력
- 카테고리 선택
- 날짜 및 시간 지정
- 메모 추가
- 영수증 첨부

**3️⃣ OCR 인식 (Document Scanning)**
- 📸 사진 촬영
- 🤖 ML Kit OCR 인식
- 💰 금액 자동 추출
- 📝 정보 자동 채우기
- ✏️ 수동 수정 옵션

**4️⃣ 반품/환불 (Refund)**
- 🔄 지출 취소 또는 반품 처리
- 💰 수입으로 기록되지만 별도 통계 처리
- 📊 '반품' 전용 아이콘(Replay) 적용 (표시는 앱 테마/유형 색상 규칙을 따름)
- 📉 지출 통계에서 차감되지 않고 별도 항목으로 집계

**카테고리**:
- 🍔 식사 (Dining)
- 🛍️ 쇼핑 (Shopping)
- 🚗 교통 (Transportation)
- 🏥 의료 (Medical)
- 📚 교육 (Education)
- 🎬 엔터테인먼트 (Entertainment)
- 💡 유틸리티 (Utilities)
- 🏠 주거 (Housing)
- 📞 통신 (Communication)
- 기타 커스텀 카테고리

**경로**: `거래 목록 → 거래 선택`

## 쇼핑 기능

### 🛒 쇼핑 카트 (Shopping Cart Screen)
**경로**: `홈 → 쇼핑 탭`

**기능**:
- 🛍️ 구매 계획 리스트 작성
- 💰 예상 비용 계산
- ✅ 구매 항목 체크(현장 구매 표시)
- 📝 항목 추가/삭제
- 📊 쇼핑 분석

**사용 방법**:
1. (쇼핑준비) 구매할 항목을 빠르게 추가
2. (장바구니) 가격/수량을 입력
3. 마트에서 물품을 담으면 품목을 탭해서 체크(구매 표시)
	- 체크된 항목은 목록 아래로 이동하여, 아직 안 산 항목이 위에 남습니다.
4. 계산대에서 반환/취소된 품목은 다시 탭하여 체크 해제(되돌림)
5. 집에 도착 후 `체크 항목 거래 입력`으로 가계부 입력 화면을 진행

### 🏪 쇼핑카트 빠른 거래 (Shopping Cart Quick Transaction)

## 🆘 문제 해결

### 일반적인 문제

**Q: 거래가 저장되지 않음**
- A: 앱 권한 확인, 저장소 용량 확인, 앱 재시작

**Q: 차트가 안 보임**
- A: 브라우저 캐시 삭제, 데이터 새로고침, 앱 업데이트

**Q: 백업 실패**
- A: 클라우드 동기화 확인, 저장소 권한 확인, 네트워크 연결 확인

**Q: 생체인증 안 됨**
- A: 생체정보 재등록, 폰 센서 확인, 설정 초기화
