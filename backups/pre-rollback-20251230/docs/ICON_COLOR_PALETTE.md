# Icon Color Palette (Original assets)

This document defines the original color palette used for sample icons (no copyrighted material).

## Palette

- Vibrant Blue
  - Primary: #0F62FE
  - Accent: #00D4FF
  - Use: action icons, primary accents

- Aqua Green
  - Primary: #00C2A8
  - Accent: #00E676
  - Use: success/asset related

- Purple Pink
  - Primary: #7C4DFF
  - Accent: #FF6EC7
  - Use: highlights, premium features

- Warm Orange
  - Primary: #FF9A3D
  - Accent: #FFD54F
  - Use: warning/attention

- Neutral Dark
  - Primary: #0F1720
  - Use: icons on light backgrounds

## Notes
- All assets are original, created for this project.
- Use gradients by pairing Primary -> Accent for depth.
- Keep contrast with text/icons to WCAG AA level.
