# 🛒 SmartLedger 쇼핑 가이드 - 빠른 시작

## 🚀 5분 안에 시작하기

### 1단계: 장바구니 준비 (2분)
```
장바구니 화면에서:
1. "우유" 입력 → 추가
2. 📍 아이콘 클릭 → "냉장고" 입력
3. 반복...
```

### 2단계: 쇼핑 안내 시작 (1초)
```
장바구니 화면에서 🗺️ 버튼 클릭!
```

### 3단계: 마트에서 사용 (실시간)
```
화면 보면서:
① 냉장고 → 우유 찾기 → ✓ 체크
② "다음 위치" 버튼
③ 3번 통로 → 커피 찾기 → ✓ 체크
④ 반복...
```

## 💡 핵심 3가지만 기억하세요

### ✅ 위치는 간단명료하게
```
❌ "냉장고 근처 어디쯤..."  
✅ "냉장고"

❌ "통로 몇 번인지 기억 안 나는데..."  
✅ "3번 통로"
```

### ✅ 한 번만 기록하면 평생 활용
```
첫 방문: "우유 - 냉장고" 기록
다음부터: 자동으로 "냉장고" 표시!
```

### ✅ 음성으로 더 편하게
```
# 장바구니 열기
"빅스비, 장바구니 보여줘"
"하이 빅스비, 장보기 목록 다 보여줘"

# 쇼핑 안내 시작
"빅스비, 쇼핑 안내 시작"

# 다음 위치 확인
"빅스비, 다음 어디 가야 돼?"
```

## 🎁 보너스 팁

### 🏠 집에서 준비
- 냉장고 확인하며 부족한 거 추가
- 각 상품 위치 입력 (1분)
- 예상 쇼핑 시간 확인

### 🚗 이동 중
- 음성으로 목록 확인
- 할인 행사 메모 추가

### 🏪 마트 입구
- 자동 루틴 설정 시 앱 자동 실행!
- 화면 보며 첫 번째 위치로 이동

### 🛒 쇼핑 중
- 찾으면 체크만 하면 됨
- 진행률 실시간 확인

### ✅ 계산 후
- 바로 가계부 입력 화면
- 포인트 적립 기록

## 🎯 목표 달성하기

### 1주 후
- ✓ 쇼핑 시간 **10분 단축**
- ✓ 놓친 상품 **0개**

### 1개월 후  
- ✓ 쇼핑 시간 **30% 이상 단축**
- ✓ 완벽한 위치 데이터 구축

### 6개월 후
- ✓ 가족 모두 사용
- ✓ 마트 쇼핑이 **즐거워짐**
